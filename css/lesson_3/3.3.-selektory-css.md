# 3.3. –°–µ–ª–µ–∫—Ç–æ—Ä—ã CSS

–¢–∞–∫ –∂–µ, –∫–∞–∫ –≤ HTML –µ—Å—Ç—å —Ç–µ–≥–∏, –≤ CSS –µ—Å—Ç—å "—Å–µ–ª–µ–∫—Ç–æ—Ä—ã".

**–°–µ–ª–µ–∫—Ç–æ—Ä—ã** ‚Äì —ç—Ç–æ "–∏–º–µ–Ω–∞" —Å—Ç–∏–ª–µ–π, –∫–æ—Ç–æ—Ä—ã–º –≤—ã –±—É–¥–µ—Ç–µ –ø—Ä–æ–ø–∏—Å—ã–≤–∞—Ç—å —Å–≤–æ–π—Å—Ç–≤–∞.

**–°–≤–æ–π—Å—Ç–≤–∞** ‚Äì —ç—Ç–æ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞, —Ç–∞–∫–∏–µ –∫–∞–∫[`color`](https://msiter.ru/references/css-reference/color),[`font-weight`](https://msiter.ru/references/css-reference/font-weight)–∏–ª–∏ [`background-color`](https://msiter.ru/references/css-reference/background-color), –∫–æ—Ç–æ—Ä—ã–º –ø—Ä–∏—Å–≤–æ–µ–Ω—ã –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è. 

–ó–Ω–∞—á–µ–Ω–∏—è —Å–≤–æ–π—Å—Ç–≤ –ø—Ä–∏—Å–≤–∞–∏–≤–∞—é—Ç—Å—è –ø—Ä–∏ –ø–æ–º–æ—â–∏ –¥–≤–æ–µ—Ç–æ—á–∏—è –∏ —Ä–∞–∑–¥–µ–ª—è—é—Ç—Å—è —Ç–æ—á–∫–æ–π —Å –∑–∞–ø—è—Ç–æ–π:

```text

   font-size: 12px;
   color: #000;
}
```

**–°–µ–ª–µ–∫—Ç–æ—Ä—ã –±—ã–≤–∞—é—Ç:**

* —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω—ã–º–∏
* —Ç–µ–≥–æ–≤—ã–π —Å–µ–ª–µ–∫—Ç–æ—Ä \(—Å–µ–ª–µ–∫—Ç–æ—Ä–æ–º —ç–ª–µ–º–µ–Ω—Ç–∞\)
* –∫–ª–∞—Å—Å–æ–≤—ã–π —Å–µ–ª–µ–∫—Ç–æ—Ä
* —Å–µ–ª–µ–∫—Ç–æ—Ä –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞
* —Å–µ–ª–µ–∫—Ç–æ—Ä –ø–æ—Ç–æ–º–∫–∞
* –¥–æ—á–µ—Ä–Ω–∏–π —Å–µ–ª–µ–∫—Ç–æ—Ä
* —Å–æ—Å–µ–¥–Ω–∏–π —Å–µ–ª–µ–∫—Ç–æ—Ä
* —Ä–æ–¥—Å—Ç–≤–µ–Ω–Ω—ã–π —Å–µ–ª–µ–∫—Ç–æ—Ä
* —Å–µ–ª–µ–∫—Ç–æ—Ä –∞—Ç—Ä–∏–±—É—Ç–∞
