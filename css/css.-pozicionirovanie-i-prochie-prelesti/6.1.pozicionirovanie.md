# 6.1.–ü–æ–∑–∏—Ü–∏–æ–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ

–°–≤–æ–π—Å—Ç–≤–æ **position** –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ç–æ—á–Ω–æ –∑–∞–¥–∞—Ç—å –Ω–æ–≤–æ–µ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ –±–ª–æ–∫–∞ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ —Ç–æ–≥–æ –º–µ—Å—Ç–∞, –≥–¥–µ –æ–Ω –Ω–∞—Ö–æ–¥–∏–ª—Å—è –±—ã –≤ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–º –ø–æ—Ç–æ–∫–µ –¥–æ–∫—É–º–µ–Ω—Ç–∞.
